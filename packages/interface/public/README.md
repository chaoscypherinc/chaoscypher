# Public Assets Folder

This folder contains static assets that are served directly by the application.

## Logo Instructions

Place your logo file here as `logo.png`:
- **File name**: `logo.png` (exactly this name)
- **Format**: PNG with transparent background
- **Aspect ratio**: Square (1:1) - e.g., 512x512px, 256x256px, or 128x128px
- **Location**: `packages/interface/public/logo.png`

The logo will appear:
- **Expanded sidebar**: Logo (40x40px) next to the app name
- **Collapsed sidebar**: Logo only (40x40px)

The image will automatically scale to fit the 40x40px display size while maintaining aspect ratio.
