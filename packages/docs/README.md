# Chaos Cypher Docs

Source for the Chaos Cypher documentation site at https://chaoscypher.com.

## Development

```bash
npm ci
npm start
```

## Build

```bash
npm run build
```

The Python API reference pages are committed as generated Markdown so the public docs build does not need access to the Python source packages.
