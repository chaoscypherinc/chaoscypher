## What This Means

- You can **use, modify, and distribute** Chaos Cypher freely
- If you **modify and deploy** Chaos Cypher as a network service, you must release your modifications under AGPL-3.0
- You can **self-host** Chaos Cypher for internal use without restriction
- **Contributing** requires signing a Contributor License Agreement (CLA) -- see [CONTRIBUTING.md](https://github.com/chaoscypherinc/chaoscypher/blob/main/CONTRIBUTING.md)

## Enterprise Edition

Chaos Cypher offers an optional **Enterprise Edition** with additional features (SSO, RBAC, audit logging, team workspaces) under a separate proprietary license. Personal-use enterprise licenses are available for free.

All direct dependencies use permissive licenses (MIT, BSD, Apache-2.0). See [ADR-0002](../architecture/adrs/0002-dependency-license-policy.md) for the dependency license policy.
